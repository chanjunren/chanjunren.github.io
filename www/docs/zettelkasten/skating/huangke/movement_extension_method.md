# movement_extension_method

> 动作拓展 — three systematic methods to create new moves from any base move: element addition, element substitution, and deconstruction-reconstruction.

## Why base moves matter

- Base (基准) moves have **clear force patterns and trajectories**
- This clarity is what makes extension possible — you know exactly what to modify
- "基准动作的发力和轨迹都很清楚，这就给后续的拓展留下许多玩味乐趣"

## Method 1: Element addition (要素添加)

Add a new element to the existing move without changing its core:
- **Brush** + pivot element → pivot brush (点规brush)
- **双鱼** + direction switching → continuous turning double fish
- **X** + jump → X-jump
- **X** + 换点 (point-switch) → Mongrel Special

The base trajectory stays intact; the addition layers on top.

## Method 2: Element substitution (要素替换)

Replace one component of the move with a different one:
- **双鱼** body-forward → body-sideways → "摆摆" (side-fish)
- Forward brush → reverse brush (swap direction)
- Change the **body orientation** while keeping the foot pattern

The structure stays; the flavor changes.

## Method 3: Deconstruction-reconstruction (解构重组)

Break the move into its atomic components, then reassemble in a new way:
- **X** is made of inner-eight + outer-eight transitions
- Rearrange: outer-eight + inner-eight in different sequence → same trajectory as a basic Nelson
- **双鱼** parallel feet + S-curve → make it all-directional → "鱼转" (fish spin)

This is the most creative method — produces moves that look completely different from the source.

## Application to any move

For any base move, ask:
1. What can I **add** without breaking the core trajectory?
2. What component can I **swap** for a different element?
3. What are the **atomic parts**, and what happens if I recombine them?

This is why learning **base/standard (基准)** versions of moves matters — they're the raw material for infinite variations. "这也是为什么我喜欢玩花桩而不是练花桩" — playing, not just practicing.
