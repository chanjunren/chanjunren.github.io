# crazy

Crazy is one of the first slalom tricks that exposes poor weight transfer — most skaters learn a working version without realising their mechanics are inefficient, because no one standardised how to break it down.

Slalom Laboratory's take: tricks consist of the same repeating sub-phases as walking (push → glide → recovery). Breaking them down makes errors visible.

## Key Concepts

- **Linkage** — smoothness between movements; the #1 thing that makes slalom look fluid
- **Weight shift** — what drives linkage; cannot be compensated by arm swinging or upper body adjustment
- The walking cycle is the direct foundation for slalom movement

## Walking Cycle → Slalom Movement

| Stage | Action                        | Phase                   |
| ----- | ----------------------------- | ----------------------- |
| 1     | Full weight on one foot       | End of weight shift     |
| 2     | Bend knee, lift trailing leg  | Start of centroid shift |
| 3     | Reach lifted foot forward     | End of centroid shift   |
| 4     | Land via forward lean         | Start of weight shift   |
| 5     | Shift weight onto landed foot | End of weight shift     |
| 6–9   | Repeat, other side            | —                       |
|       |                               |                         |

Even though walking feels natural, there are efficient and inefficient ways to do it — same as skating. Bad weight shift can go unnoticed because the overall visual trace still looks normal.

Weight shift is the highest-priority thing to get right. Everything else (arms, posture, head) is secondary.
